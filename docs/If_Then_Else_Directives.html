@If Directive

Lets you compile some code and ignore other code for different conditions.

Syntax

@If condition [ Then ]
    [ statements ]
[ @Else
    [ statements ] ]
@End [ If ]

Parts

condition
Mandatory
statements
Optional declarations or executable statements.
@Else
Optional
@End
Completes the directive block. You can also use @End If.
