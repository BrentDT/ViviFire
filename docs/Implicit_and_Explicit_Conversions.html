Automatic and Non-Automatic Conversions

An automatic conversion lets the compiler change a data type automatically. No special syntax is necessary. In the example that follows, ViviFire automatically changes the value of k to a floating-point value, then assigns that to q.

Dim k As Int
Dim q As Real
' Int widens to Real
k = 321
q = k

A non-automatic conversion uses an operator to change a data type to a different one. ViviFire supplies two such operators, As and In, which change an expression to the necessary data type.

The example below continues from the example before. The value of q is changed to Int, then assigned to k.

' q has the value 321.
q = Math.Sqrt(q)
k = q As Int
' k has the value 18 (rounded square root of 321).

In operator

The operator In can change a numeric expression to a unit of measure. If the expression is a unit of measure, then it can change it to a different unit from the same base unit. All units of measure are stored as floating-point types.
