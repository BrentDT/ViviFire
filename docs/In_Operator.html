In Operator

Gives a numeric value a unit of measure.

Syntax

expression In unit

Parts

expression
A numeric expression.
unit
The name of a unit of measure.

Instructions

You use the operator In when an expression must become a unit of measure, such as an argument to a procedure with that unit.

The operator is not necessary when the expression is of a compatible unit, then it changes automatically. It is also not necessary after a simple numeric literal, where only the unit is necessary.

If you try to change a value in one unit to one that is not compatible, then the result is a compiler error.
