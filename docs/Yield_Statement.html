Yield Statement

Sends an element of a collection to a loop of the type For Each.

Syntax

Yield expression

Parts

expression
Mandatory value to send to the loop.

Instructions

The data type of expression must be compatible with the return type of the procedure.

Yield is not permitted in blocks of the type Finally.
