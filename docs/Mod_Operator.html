Mod Operator

Divides two numbers and returns the remainder.

Syntax

expression_1 Mod expression_2

Parts

expression_1
A numeric expression.
expression_2
A numeric expression.

Instructions

The result has the same sign as the first expression. Compared to Mod, the result of the related operator Rem has the sign of the second expression. It is easy to select the correct operator if you see that the letter “M” is nearer the expression that supplies the sign.
