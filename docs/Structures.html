Structures

A structure is a user-defined type that can hold several values at the same time. Each value is known as a field and has a name. Structures can also have properties, methods, and events. And a structure can be an implementation of one or more traits.

You can put together data of different types in one structure. The elemets become related to the structure. Structures that you declare are data types that you can use to declare variables.
