Call Base Statement

Lets a constructor in a child class call a constructor in a parent class.

Syntax

Call Base [ argument_list ]

Parts

argument_list
Optional one or more expressions, with a comma between each. You can put parentheses around the list, if you want.

Examples

@Open Class BaseClass
    Var a As Int
    Var b As Real

    Constructor
        a = 0
        b = 0.0
    End Constructor

    Constructor x As Int, y As Real
        a = x
        b = y
    End Constructor
End Class

Class DerivedClass Is BaseClass
    Constructor
        Call Base
    End Constructor

    Constructor a As Int, b As Real
        Call Base a, b
    End Constructor
End Class
