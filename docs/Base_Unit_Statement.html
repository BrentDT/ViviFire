Base Unit Statement

Supplies a physical quantity with a standard measure.

Syntax

Base Unit name In standard_unit [ = expression ]

Parts

name
Mandatory name for a physical quantity, such as length or time.
standard_unit
Mandatory unit expression is the standard unit to measure the given physical quantity.
expression
Optional unit expression makes a relation between a physical quantity and its parents.

Instructions

The names used in standard_unit must come from declarations of Unit. The names used must not be the same as some different units. For derived units, you can use expression to prevent this problem.

You can use name to declare elements, such as variables. Then it uses the standard unit. But you cannot use name in unit expressions, such as those after a number.

Examples

Simple units

Base Unit length In meters
Base Unit mass In kilograms
Base Unit time In seconds

Derived units

Base Unit velocity In (meters/second) = (length/time)
