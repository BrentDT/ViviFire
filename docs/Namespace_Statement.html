Namespace Statement

Declares the name of a namespace and causes the compiler to include the code that follows in that namespace.

Note: This part of the language will possibly be removed. Use a module with the keyword Library as an alternative.

Syntax

Namespace [ Global . ] qualified_name
    [ statements ]
End [ Namespace ]

Parts

Global
Optional name lets you define a namespace that is external to your project.
qualified_name
Mandatory qualified name. That is a list of one or more names with a dot (.) between each.
statements
Optional one or more declarative statements.
End
Completes the statement. You can also use End Namespace.
