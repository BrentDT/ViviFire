Optional

Specifies that you can ignore some parameters when you call a procedure.

Instructions

For each optional parameter, you can give a constant expression as the default value of that parameter. If you give it the value #Null or do not supply a default value, then ViviFire uses the default value for the data type of the parameter. If you use the directive @Option Optional With Equals, then you must supply a default value.

If the parameter list contains an optional parameter, then all parameters after it are also optional. If you use Optional more than one time, then the compiler usually shows a warning.

Applies to

  • Declare Statement
  • Function Statement
  • Method Statement
  • Sub Statement
