How to Declare a Structure

You start a structure declaration with Struct, and you complete it with End. You must write each of these on different lines. Between these lines you must declare one or more elements. All data types are permitted, but not all can have @Shared for a modifier.

You can initialize elements in the structure declaration the same as you do with Dim.

When you declare a variable with a structure type, you get access to the elements through the variable.

To compare structures with classes, see Structures and Classes.
