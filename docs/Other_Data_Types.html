Other Data Types

ViviFire supplies some data types that are not related to numbers or characters. They let you do work on special data such as yes/no values and date/time values.

For a table to compare all the different data types, see Data Type Summary.

Boolean type

The Boolean Data Type is an unsigned integer that ViviFire interprets as one of True or False. Its width can be different on different platforms. If you must have a variable with two values (for example, true/false, yes/no, or on/off), then declare it as Boolean.

DateTime type

The DateTime Data Type is a 64-bit integer that holds date and time values. Each increment is 100 nanoseconds and starts at midnight of the first of January of the year 1 in the Gregorian calendar. If a variable must hold a date, a time, or the two, then declare it as DateTime.
