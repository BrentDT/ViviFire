/ Operator

Divides one number by a second number.

Syntax

expression_1 / expression_2

Parts

expression_1
A numeric expression.
expression_2
A numeric expression.

Instructions

Data types

If the two expressions are integers, then they automatically change to Real, and the result is Real.
