Modifiers

A modifier is a special word that can change the operation of a statement.

ViviFire has two different types of modifiers. Modifiers that you write before a statement start with the symbol «@». Modifiers in other positions are the usual reserved words.

Many modifiers apply to several statements. Articles in this section usually have a list with the heading “Applies to” that gives this information.

In this section

  • @Abstract
  • ByRef
  • @Const
  • @Deprecated
  • In
  • @Iterator
  • @MustUse
  • @Open
  • Optional
  • Out
  • @Override
  • @Shared

Other modifiers

Some modifiers apply to only one statement. What follows is a list of such modifiers with a link to the applicable statement.

  • @Backed (Property)
  • @Flags (Enum)
  • @ReadOnly (Property)
  • @SI (Unit)
  • @Unique (Enum)
  • @WriteOnly (Property)
