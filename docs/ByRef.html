ByRef

Specifies that a parameter of a procedure, when passed a variable as an argument, makes an alias of the variable. This lets the procedure change the variable in the caller's scope.

Applicable to

  • Declare Statement
  • Function Statement
  • Method Statement
  • Sub Statement
